## Host Room-Monster
<ul>
   <li>
      <h4>In the HOST ROOM, the host can select the style of the Monster and the game mode. There are three Monster roles and two different map difficulties to choose from.</h4>
   </li>
   <li>
      <h4>Initially, the chat room displays a 'waiting' avatar. Once a user clicks 'Ready', the avatar changes to the selected character.</h4>
   </li>
    <li>
      <h4>Players can communicate with each other within the chat room.</h4>
   </li>
    <li>
      <h4>The host room needs to provide the room code to other players.</h4>

## Game Map

<ul>
   <li>
      <h4>Aliens have the ability to place and break blocks, but they cannot destroy indestructible obstacles.</h4>
   </li>
   <li>
      <h4>Monsters are unable to place blocks, but they can break all types of blocks, including indestructible obstacles.</h4>
   </li>
</ul>

## LeaderBoard

<ul>
   <li>
      <h4>If monsters capture all aliens, the monsters win.</h4>
   </li>
   <li>
      <h4>Aliens achieve victory by collecting all energy sources.</h4>
   </li>
    <li>
      <h4>In case the time runs out and the monsters have not captured all aliens, and the aliens have not collected all energy sources, the game ends in a draw.</h4>

## Installation Guide

<ul>
   <li>
      <h4>Clone the Repository</h4>
      <p>First, clone the repository to your local machine using Git.</p>
   </li>
   <li>
      <h4>Install Client Dependencies</h4>
      <p>Navigate to the client directory and run <code>npm install</code> to install all necessary dependencies for the client side.</p>
   </li>
   <li>
      <h4>Install Server Dependencies</h4>
      <p>Similarly, navigate to the server directory and run <code>npm install</code> to install dependencies for the server side.</p>
   </li>
   <li>
      <h4>Start the Application</h4>
      <p>Start the client application: <code>npm run start</code></p>
       <p>Start the server: <code>npm start</code></p>
   </li>
</ul>
